float Tree::lifeOdds ( float  height,
float  soil,
float  flowmap 
)

Renvoie la probabilité d'existence de l'arbre.

La probabilité d'existence correspond aux chances d'un arbre d'exister selon un certain nombre de conditions de terrain.

Paramètres
height: Altitude du terrain.
soil: Hauteur de la couche de sédiments.
flowmap: Taux d'humidité.
Renvoie
Une probabilité entre 0 et 1.
